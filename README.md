# OSINT Phishing Domain & Malicious URL Investigation

This project demonstrates a structured Open Source Intelligence (OSINT) investigation of a phishing URL hosted on a trusted third-party platform.

## Objective
To analyze a suspected phishing URL using publicly available OSINT techniques and identify indicators of credential harvesting activity.

## Investigation Summary
- Identified phishing activity hosted on a user-created Blogspot subdomain
- Analyzed DNS, IP, ASN, and hosting infrastructure
- Confirmed malicious classification using VirusTotal
- Documented findings in a formal OSINT investigation report

## Key Findings
- Abused Platform: Blogspot (Google Blogger)
- IP Address: 192.178.155.132
- ASN: AS15169 (Google LLC)
- Threat Type: Phishing (Credential Harvesting)
- VirusTotal Detection Ratio: 15/93

## Skills Demonstrated
- Open Source Intelligence (OSINT)
- Threat Intelligence Analysis
- Phishing Detection
- DNS & Hosting Analysis
- Security Research & Documentation

## Disclaimer
This project was conducted strictly for educational and research purposes using publicly available information. No interaction with malicious infrastructure was performed.
